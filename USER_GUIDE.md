# Keep Fit 应用访问指南

欢迎使用 Keep Fit 健身追踪应用！

**应用地址**：https://mkshi77.github.io/keep-fit/

---

## 📱 如果无法访问，请尝试以下方法

### 方法 1：修改 DNS 设置（推荐）⭐

#### 手机设置步骤：

**小米手机**：
1. 设置 → WLAN
2. 长按已连接的 WiFi → 修改网络
3. 高级选项 → IP 设置改为"静态"
4. DNS 1 填写：`223.5.5.5`（阿里云 DNS）
5. DNS 2 填写：`8.8.8.8`（Google DNS）
6. 保存后重新访问

**华为手机**：
1. 设置 → WLAN
2. 点击已连接的 WiFi 右侧的 ">"
3. 静态 IP → DNS 1：`223.5.5.5`，DNS 2：`8.8.8.8`
4. 保存后重新访问

**iPhone**：
1. 设置 → 无线局域网
2. 点击已连接 WiFi 右侧的 "i"
3. 配置 DNS → 手动
4. 添加服务器：`223.5.5.5` 和 `8.8.8.8`
5. 保存后重新访问

---

### 方法 2：使用移动数据网络

有时候 4G/5G 比 WiFi 访问更稳定：
1. 关闭 WiFi
2. 使用移动数据访问
3. 如果能打开，说明是 WiFi 网络问题

---

### 方法 3：更换浏览器

**推荐浏览器**（按优先级）：
1. ✅ Chrome（最推荐）
2. ✅ Microsoft Edge
3. ✅ Firefox
4. ⚠️ Safari（iPhone 自带）
5. ⚠️ 手机自带浏览器（可能有限制）

---

### 方法 4：清除浏览器缓存

1. 浏览器设置 → 隐私/安全
2. 清除浏览数据/缓存
3. 重启浏览器后访问

---

### 方法 5：稍后重试

GitHub Pages 在国内访问时好时坏，可以：
- 换个时间段访问（如晚上或早上）
- 多刷新几次
- 等待 5-10 分钟后重试

## ✅ 应用功能说明

### 数据存储
- ✅ 所有数据保存在**手机浏览器本地**
- ✅ 不会上传到服务器
- ✅ 换设备或浏览器数据不同步
- ⚠️ 清除浏览器数据会丢失记录

### 每日重置
- 每天首次打开会重置当日任务
- 历史记录和体重数据会保留

### 视频播放
- 训练视频会自动播放
- 如果没有播放，点击屏幕任意位置

## ❓ 常见问题

### Q: 为什么有时能访问，有时不能？
**A**: GitHub Pages 在国内访问不稳定，受网络环境影响。建议修改 DNS 或使用移动数据。

### Q: 数据会丢失吗？
**A**: 只要不清除浏览器数据，记录会一直保存。建议定期截图备份重要数据。

### Q: 可以在多个设备使用吗？
**A**: 可以，但数据是独立的，不会自动同步。

### Q: 视频不播放怎么办？
**A**: 点击屏幕任意位置激活播放，或检查浏览器是否禁用了自动播放。
